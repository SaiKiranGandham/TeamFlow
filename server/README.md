# TeamFlow

## Project Overview
TeamFlow is a team task management application that helps users manage projects, tasks, and team activities.

## Technologies Used

Frontend:
- React
- React Router

Backend:
- Node.js
- Express

## Setup Instructions

### Frontend

Open terminal:

cd client

Install packages:

npm install

Run:

npm start


### Backend

Open another terminal:

cd server

Install packages:

npm install

Run:

node server.js


## Environment Variables Required

No environment variables are required for this implementation.

## Assumptions Made

- Users can view and manage tasks through the application.
- The application is designed as a task management prototype.
- Authentication is represented as a basic UI flow.

## Features Implemented

- Home page
- Login page
- Dashboard
- Task management page
- Backend API
- Frontend and backend integration
- Navigation using React Router

## Known Limitations

- User authentication is not connected to a database.
- Tasks are not permanently stored.
- No role-based access control.
